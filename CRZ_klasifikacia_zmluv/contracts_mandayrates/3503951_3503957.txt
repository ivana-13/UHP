Článok I.
DEFINÍCIE POJMOV

1.1

Web dokument – dokument s obsahom Objednávateľa, ktorý je umiestnený a
sprístupnený v sieti internet pod dohodnutou doménou.

1.2

Web sídlo – všetky Web dokumenty umiestnené spravidla pod jednou doménou,
vrátane akýchkoľvek aplikačných riešení prevádzkovaných na internetovej alebo
intranetovej sieti a dostupných cez Web prehliadač, pričom v tejto Zmluve sa pod
týmto pojmom rozumie Web sídlo Objednávateľa, pokiaľ výslovne z textu
ustanovenia Zmluvy nevyplýva, že sa jedná o Web sídlo tretej osoby.

1.3

Web prehliadač – program na počítači Objednávateľa, umožňujúci prezeranie Web
dokumentov.

1.4

Doména – internetová adresa, na ktorej je prevádzkované a prístupné Web sídlo.

1.5

Server – technologické zariadenie, hardvérové a softvérové riešenie, ktoré umožňuje
prevádzkovať Web sídlo.

1.6
Provider – tretia strana, ktorá zabezpečuje pripojenie, konektivitu Servera na sieť
internet.
1.7
Redakčný systém – nástroj, produkt Dodávateľa umožňujúci aktualizáciu a správu
Web sídla Objednávateľom.
1.8

SwiftSite – meno redakčného systému, ktorý vytvoril a vlastní Dodávateľ.

1.9

Akceptačný protokol – písomné potvrdenie Objednávateľa o akceptovaní funkčnosti
a spôsobilosti Web sídla na jeho riadne spustenie.

ktoré bránili v odovzdaní diela a Dodávateľ je povinný ich bezodkladne na vlastné
náklady odstrániť.

bude dielo funkčné, bude sa Dielo považovať za dodané siedmym dňom po oznámení
Dodávateľa o umiestnení Web sídla na testovaciu adresu.

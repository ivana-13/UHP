Článok I
VYMEDZENIE ZÁKLADNÝCH POJMOV
1.

Web dokument - Dokument s obsahom Objednávateľa, ktorý je umiestnený a
sprístupnený na internete pod dohodnutou doménou.

2.

Web sídlo – súhrn Web dokumentov umiestnených spravidla pod jednou doménou,
vrátane akýchkoľvek aplikačných riešení prevádzkovaných na internetovej alebo
intranetovej sieti a dostupných cez Web prehliadač, pričom v tejto Zmluve sa pod týmto
pojmom rozumie Web sídlo vytvorené a dodané na základe Zmluvy o vytvorení a
prevádzke web sídla a o poskytovaní služieb zo dňa 19. 10. 2007 Objednávateľovi
Dodávateľom, pokiaľ zo Zmluvy nevyplýva, že ide o Web sídlo inej osoby.

3.

Web prehliadač - Program na počítači Objednávateľa, umožňujúci prezeranie Web
dokumentov.

4.

Doména - Internetová adresa Objednávateľa, na ktorej je prevádzkované a prístupné
Web sídlo.

5.

Server - Technologické zariadenie, hardvérové a softvérové riešenie, ktoré umožňuje
prevádzkovať Web sídlo, nachádzajúce sa u Dodávateľa, pokiaľ zo Zmluvy nevyplýva, že
ide o Server inej osoby.

6.

Provider - Tretia strana, ktorá zabezpečuje pripojenie, konektivitu Servera na sieť
internet.

7.

Redakčný systém - Nástroj, produkt Dodávateľa umožňujúci aktualizáciu a správu Web
sídla Objednávateľom.

8.

SwiftSite™ Profi - Názov redakčného systému, ku ktorému Dodávateľ vykonáva
majetkové práva autora.

9.

Webhosting - umiestnenie a prevádzkovanie Web sídla na Serveri Dodávateľa.

10.

Zákazník - Akákoľvek tretia osoba, ktorá má záujem o Web dokumenty a ďalší obsah
nachádzajúci sa na Web sídle Objednávateľa.
